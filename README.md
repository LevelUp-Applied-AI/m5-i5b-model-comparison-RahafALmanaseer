# Integration 5B — Model Comparison & Decision Memo

Module 5 culminating deliverable for AI.SPIRE Applied AI & ML Systems.

## Setup

```bash
pip install -r requirements.txt
```

## Tasks

Build a complete model comparison pipeline with 6 model configurations, PR curves, calibration plots, experiment logging, and model persistence. Write a decision memo recommending a model for Petra Telecom. See the integration guide for detailed instructions.

## Submission

Your PR description must include:
1. Model comparison table (all 6 models × 5 metrics)
2. Decision memo (2–3 paragraphs)
3. Paste your PR URL into TalentLMS → Module 5 Week B → Integration 5B to submit this assignment
